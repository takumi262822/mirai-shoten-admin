== 概要

本書はプロジェクト「{project}」で利用する定数値を列挙したものです。
ソースファイル `src/constants/admin-constants.js` から逆引きで抽出しています。

---

== バリデーション規則（VALIDATION）

[cols="2,1,1,4", options="header"]
|===
|定数名
|型
|値
|内容

|VALIDATION.MIN_CUSTOMER_NAME
|number
|1
|顧客名の最小文字数

|VALIDATION.MAX_CUSTOMER_NAME
|number
|50
|顧客名の最大文字数

|VALIDATION.MIN_QUANTITY
|number
|1
|注文数量の最小値

|VALIDATION.MAX_QUANTITY
|number
|999
|注文数量の最大値

|VALIDATION.MIN_PRICE
|number
|0
|合計金額の最小値（0 円は許容）

|VALIDATION.MAX_PRICE
|number
|999999
|合計金額の最大値

|===

== ストレージキー（STORAGE_KEYS）

[cols="2,1,2,4", options="header"]
|===
|定数名
|型
|値
|内容

|STORAGE_KEYS.ADMIN_ORDERS
|string
|"adminOrders"
|注文一覧を永続化する localStorage キー

|STORAGE_KEYS.FUTURE_SHOP_CART
|string
|"futureShopCart"
|未来商店商材のカートデータと共有する localStorage キー（読込専用）

|===

== UI メッセージ（UI_MESSAGES）

[cols="2,1,3,4", options="header"]
|===
|定数名
|型
|値
|内容

|UI_MESSAGES.ORDER_ADDED
|string
|注文を追加しました
|注文追加成功時の通知メッセージ

|UI_MESSAGES.ORDER_UPDATED
|string
|注文を更新しました
|注文更新成功時の通知メッセージ

|UI_MESSAGES.ORDER_DELETED
|string
|注文を削除しました
|注文削除成功時の通知メッセージ

|UI_MESSAGES.INVALID_EMAIL
|string
|有効なメールアドレスを入力してください
|メール検証エラー時のメッセージ

|UI_MESSAGES.INVALID_QUANTITY
|string
|数量は1〜999の整数です
|数量検証エラー時のメッセージ

|UI_MESSAGES.INVALID_STATUS
|string
|有効なステータスを選択してください
|ステータス検証エラー時のメッセージ

|UI_MESSAGES.REQUIRED_FIELD
|string
|この項目は必須です
|必須フィールド未入力時のメッセージ

|UI_MESSAGES.CONFIRM_DELETE
|string
|本当に削除しますか？
|削除確認ダイアログの表示文言

|===

== 概要

本書はプロジェクト「{project}」に登場するコード値（識別子・種別値）を列挙したものです。
ソースファイル src/constants/admin-constants.js から逆引きで抽出しています。

---

== 注文ステータスコード（ORDER_STATUS）

ファイル: src/constants/admin-constants.js

[cols="1,3,3,2,1,4,3", options="header"]
|===
|項番
|コード和名
|コード英名
|データ型
|桁数
|説明
|記述形式

|1
|受付
|pending
|文字型
|7
|注文受付済み・処理前の状態。注文一覧バッジはオレンジ（#ff9900）で表示
|半角英小文字

|2
|処理中
|processing
|文字型
|10
|注文処理中（確認・在庫引当中）。バッジはブルー（#3366cc）で表示
|半角英小文字

|3
|発送済み
|shipped
|文字型
|7
|発送済み・輸送中の状態。バッジはディープブルー（#0066cc）で表示
|半角英小文字

|4
|配達完了
|delivered
|文字型
|9
|顧客への配達完了。バッジはグリーン（#009900）で表示
|半角英小文字

|5
|キャンセル
|cancelled
|文字型
|9
|注文キャンセル済み。バッジはグレー（#999999）で表示
|半角英小文字

|===

== 操作仕様

[#操作仕様, title="操作仕様", cols="2,5", options="header", align="center"]
|===
^.^|操作 ^.^|仕様

<.^|カートへの追加 <.^|`localStorage.getItem('futureShopCart')` を取得し JSON.parse する。既存レコードに同一 `id` があれば `quantity` を加算。なければ商品オブジェクトを `push` して保存する
<.^|合計金額の計算 <.^|`CartService.calculateTotal(cart)` で計算する。`price × quantity` の総和を返す
<.^|カートクリア <.^|購入完了後に `localStorage.removeItem('futureShopCart')` でカートを空にする
|===

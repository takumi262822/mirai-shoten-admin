== カラム定義

[#カラム定義, title="カラム定義", cols="1,3,3,2,1,1,3,4", options="header", align="center"]
|===
^.^|No. ^.^|カラム名（論理名） ^.^|カラム名（物理名） ^.^|型 ^.^|必須 ^.^|主キー ^.^|デフォルト値 ^.^|備考

^.^|{counter:カラム} <.^|注文ID <.^|id <.^|string <.^|○ <.^|PK <.^|- <.^|`ORD-$\{Date.now()}-$\{Math.random().toString(36).slice(2,8)}` 形式で自動採番。旧形式データも `normalizeOrderRecord` で正規化する
^.^|{counter:カラム} <.^|顧客名 <.^|customerName <.^|string <.^|○ <.^|- <.^|`'未設定'` <.^|1〜50 文字。使用可能文字: 英数字・日本語（ひらがな・カタカナ・漢字）・スペース。XSS サニタイズ済みで保存
^.^|{counter:カラム} <.^|メールアドレス <.^|email <.^|string <.^|○ <.^|- <.^|`'未設定'` <.^|RFC 準拠の形式チェックあり。全角ASCII は半角に正規化して保存。XSS サニタイズ済み
^.^|{counter:カラム} <.^|商品コード <.^|productCode <.^|string <.^|○ <.^|- <.^|`'UNSPECIFIED'` <.^|大文字変換済みで保存（`toUpperCase()`）。旧形式の `items` キーも吸収する
^.^|{counter:カラム} <.^|数量 <.^|quantity <.^|number <.^|○ <.^|- <.^|`1` <.^|1〜999 の整数。不正値（NaN・0以下）は `1` にフォールバック
^.^|{counter:カラム} <.^|合計金額 <.^|totalPrice <.^|number <.^|○ <.^|- <.^|`0` <.^|0以上の数値。カンマ等の非数値文字を除去してから変換する。不正値は `0` にフォールバック
^.^|{counter:カラム} <.^|注文ステータス <.^|status <.^|string <.^|○ <.^|- <.^|`'pending'` <.^|`AdminConstants.ORDER_STATUS` で定義された5値のいずれか（下記「ステータス定義」参照）。不正値は `'pending'` にフォールバック
^.^|{counter:カラム} <.^|作成日時 <.^|createdAt <.^|string <.^|○ <.^|- <.^|- <.^|ISO 8601 形式（例: `2026-04-06T09:00:00.000Z`）。旧形式の `date` キーも吸収して正規化する
^.^|{counter:カラム} <.^|更新日時 <.^|updatedAt <.^|string <.^|○ <.^|- <.^|- <.^|ISO 8601 形式。ステータス変更・内容更新のたびに `new Date().toISOString()` で上書きする
|===

== バリデーション制約

`OrderValidator.isValidOrderData(orderData)` で一括検証する。

[#バリデーション制約, title="バリデーション制約", cols="2,3,3,4", options="header", align="center"]
|===
^.^|カラム名（物理名） ^.^|制約種別 ^.^|制約内容 ^.^|エラーメッセージ定数

<.^|customerName <.^|必須チェック <.^|空文字・null・undefined を不可 <.^|`REQUIRED_FIELD`
<.^|customerName <.^|長さチェック <.^|1〜50 文字（`MIN_CUSTOMER_NAME` 〜 `MAX_CUSTOMER_NAME`） <.^|`顧客名は1〜50文字です`
<.^|customerName <.^|文字種チェック <.^|英数字・日本語・スペースのみ（`/^[a-zA-Z0-9\u3040-\u9fff\s]*$/`） <.^|`顧客名に使用できない文字があります`
<.^|email <.^|形式チェック <.^|RFC 準拠の正規表現でチェック <.^|`INVALID_EMAIL`
<.^|quantity <.^|範囲チェック <.^|1〜999 の整数（`MIN_QUANTITY` 〜 `MAX_QUANTITY`） <.^|`INVALID_QUANTITY`
<.^|totalPrice <.^|範囲チェック <.^|0〜999999 の数値（`MIN_PRICE` 〜 `MAX_PRICE`） <.^|-
<.^|status <.^|列挙値チェック <.^|`AdminConstants.ORDER_STATUS` の5値のいずれか <.^|`INVALID_STATUS`
|===
